# Packages

Здесь лежат переиспользуемые пакеты проекта.

## Структура

- `api` — backend и доменная логика системы
- `shared` — общие типы, константы и утилиты
- `ui` — общий UI-слой и дизайн-элементы для web

## Почему так

Такой подход позволяет не дублировать код между `Next.js` приложением и Telegram-ботом и держать бизнес-логику вне UI.
