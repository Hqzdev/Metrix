# Shared Package

Этот пакет содержит код, который используется и сайтом, и ботом, и backend-слоем.

## Что здесь будет

- общие типы;
- константы;
- утилиты без привязки к платформе;
- общие DTO и helper-функции.

## Границы пакета

`shared` должен оставаться безопасным для импорта из web, bot и backend-кода.
Сюда не добавляют зависимости от Node-only API, Prisma, Redis, Telegram SDK или React.
