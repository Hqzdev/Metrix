# UI Package

Этот пакет содержит общий UI-слой для web-приложения.

## Что здесь будет

- базовые компоненты интерфейса;
- дизайн-токены;
- общие стилевые примитивы;
- элементы таблиц, форм, карточек и фильтров.

## Границы пакета

`ui` отвечает только за представление. Компоненты не должны напрямую обращаться к API,
Prisma, Redis или Telegram-слою; данные передаются через props и callbacks.
