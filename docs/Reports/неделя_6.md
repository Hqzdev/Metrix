**Неделя:** 6 из 6  
**Дата составления отчёта:** 9 июня 2026 г.

## КТО ЧТО ДЕЛАЛ

**Ярослав** — реализовал WebSocket Gateway, настроил CI/CD pipeline (GitHub Actions), провёл нагрузочное тестирование (k6, p95 = 210 мс), выполнил production-деплой на Vercel

**Алексей** — провёл ручное тестирование Telegram-бота, составил баг-репорт (4 дефекта)

**Мухаммад** — стабилизировал CI/CD: починил сборку API, Bot microservices и contract-тесты, закрыл Dependency scanning; обновил документацию по тестированию, очередям и booking-сервису; провёл Lighthouse-аудит (Performance 92, SEO 97) и smoke-тестирование после деплоя

**Даниил** — добавил unit-тесты и refactor payment-саги; настроил semantic-release, changelog automation и GitHub Actions workflow для CD; расширил документацию бота, мониторинга и deployment-процесса; исправил SSH-деплой в CD pipeline

**Ратмир** — добавил TypeDoc/JSDoc и настроил генерацию API reference docs; добавил lint-job для web и нагрузочный тест на concurrent booking; исправил мобильное меню (a11y) и нормализовал GHCR-деплой; обновил документацию по контрактам, web и tooling
