Telegram Bot Diagrams

Эта папка содержит Mermaid-диаграммы.

Что здесь есть

- общий обзор бота;
- booking flow;
- cancel/reschedule flow;
- reminder flow;
- microservices architecture;
- database schema;
- Redis Streams;
- service HTTP API.

Как использовать

Открыть .mmd файл в редакторе с Mermaid preview.

Важно

Диаграммы не заменяют код.
Если flow изменился, обнови diagram.
