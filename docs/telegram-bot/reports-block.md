Reports Block

Этот документ объясняет отчеты.

Зачем нужны отчеты

Администратор хочет понимать:

- сколько было броней;
- какие ресурсы популярны;
- какие часы самые загруженные;
- где есть проблемы.

Как это должно работать

1. Пользователь или админ запрашивает отчет.
2. Система создает Report record.
3. Worker генерирует отчет.
4. Статус обновляется.
5. Админ получает файл или ссылку.

Почему через worker

Генерация отчета может быть долгой.
HTTP-запрос не должен ждать тяжелую работу.
