Telegram Bot commands

Этот документ описывает команды Telegram-бота.
Сейчас команды находятся не в отдельной папке commands, а в apps/bot/services/bot-gateway/src/bot.ts.

Назначение

Команды — это короткие сообщения, которые пользователь пишет боту вручную.
Например: /start или /book.

Команда не делает всю работу сама.
Она только запускает нужный сценарий, а дальше bot-gateway показывает кнопки и обращается к внутренним сервисам.

Список команд

/start

Открывает главное меню.
Если у пользователя был незаконченный сценарий, состояние сбрасывается в START.

/help

Показывает справку и главное меню.

/book

Начинает бронирование.
Бот переводит пользователя в состояние SELECT_LOCATION и показывает список локаций.

/slots

Работает почти как /book.
Команда открывает путь к просмотру свободных слотов через выбор локации и комнаты.

/resume

Продолжает сценарий, если пользователь остановился на середине.

Например, если пользователь уже выбрал локацию, бот снова покажет выбор комнаты.
Если сохранённого состояния нет, бот покажет главное меню.

/my_bookings

Показывает активные бронирования пользователя.
Если бронирования есть, рядом с ними появляются кнопки отмены.

/calendar

Показывает статус подключения календаря.
Если Google Calendar не подключён, бот показывает кнопку для подключения.
Если календарь уже подключён, бот показывает статус и кнопку отключения.

/stats

Показывает статистику бронирований.
Команда доступна только администраторам из ADMIN_TELEGRAM_IDS.

Как команда обрабатывается

1. Пользователь отправляет текст в Telegram.
2. bot-gateway получает message update.
3. Метод handleMessage читает текст.
4. Если текст совпал с командой, запускается нужный сценарий.
5. bot-gateway отправляет сообщение пользователю или вызывает внутренний сервис.

Что происходит после команды

После большинства команд пользователь продолжает работу кнопками.

Например:

1. /book открывает список локаций.
2. Пользователь нажимает локацию.
3. Бот показывает комнаты.
4. Пользователь нажимает комнату.
5. Бот показывает свободное время.
6. Пользователь выбирает слот.
7. Бот просит подтвердить бронь.
8. После подтверждения создаётся счёт на оплату.

Где регистрируются команды

Список команд для меню Telegram задаётся в TelegramClient.setMyCommands.
Этот метод находится в apps/bot/services/bot-gateway/src/telegram-client.ts.

Если добавить новую команду, её нужно добавить в два места:

1. в handleMessage внутри bot.ts
2. в setMyCommands внутри telegram-client.ts

Если команда открывает новую кнопку, также нужно обновить:

- keyboards.ts — чтобы добавить кнопку
- callback-data.ts — чтобы описать данные кнопки
- messages.ts — если нужен новый текст сообщения

Важное правило

Команды должны оставаться простыми.
Они не должны напрямую менять базу данных.
Если нужно создать бронь, отменить бронь, получить календарь или статистику, bot-gateway должен вызвать соответствующий сервис.
