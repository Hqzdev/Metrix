Telegram Commands Block

Этот документ объясняет команды Telegram-бота.

Основные команды

/start — начать работу.
/help — показать помощь.

Основная навигация идет через inline buttons.

Что делает command layer

- принимает текстовую команду;
- определяет действие;
- показывает меню;
- передает работу во внутренние сервисы.

Что здесь не должно быть

- прямой работы с PostgreSQL;
- прямой бизнес-логики платежей;
- сложной логики бронирования.

Команды должны быть тонким UI-слоем.
