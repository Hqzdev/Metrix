Telegram Lib Block

Этот документ объясняет вспомогательный bot-код.

Что такое lib block

Это общие функции, которые помогают bot-gateway:

- собрать сообщение;
- разобрать callback data;
- вызвать внутренний сервис;
- обработать состояние пользователя;
- сформировать клавиатуру.

Главное правило

lib code должен помогать UI-слою.
Он не должен становиться скрытым местом для бизнес-логики бронирования или оплаты.
