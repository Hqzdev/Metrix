Calendar Integrations

Этот документ объясняет календарные интеграции.

Зачем нужны календари

Когда пользователь бронирует ресурс, событие можно синхронизировать с календарем.

Google Calendar

Google flow использует OAuth.
Refresh token хранится зашифрованным.

Microsoft Calendar

Архитектура предусматривает Microsoft Calendar.
Его нужно подключать по тем же правилам безопасности.

OAuth state

State подписывается, чтобы его нельзя было подменить.

Главное правило

Календарь — внешняя система.
Она может быть недоступна.
Поэтому ошибки календаря не должны ломать сохраненную бронь.
