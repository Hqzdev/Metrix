Telegram Bot Block

Этот документ объясняет сам Telegram bot UX.

Что делает бот

Бот помогает пользователю:

- посмотреть локации;
- выбрать ресурс;
- выбрать слот;
- создать бронь;
- оплатить, если нужно;
- посмотреть свои брони;
- отменить бронь.

Главный сервис

bot-gateway.

Что хранится в Redis

- состояние диалога пользователя;
- processed update ids;
- polling offset;
- rate limit counters.

Почему это важно

Telegram может прислать update повторно.
Бот должен понять, что update уже обработан.
