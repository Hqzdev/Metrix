Screenshots

Эта папка хранит screenshots для evidence.

Какие screenshots нужны

- Docker compose ps;
- health checks;
- readiness checks;
- metrics endpoint;
- Prometheus targets;
- Prometheus alerts;
- Grafana dashboard;
- logs viewer;
- DLQ replay;
- restore drill evidence, если есть UI/terminal screenshot.

Важно

Не добавлять красивые placeholder screenshots.
Screenshot должен доказывать реальную проверку.
