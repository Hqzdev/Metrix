# Telegram Bot

Это отдельное приложение Telegram-бота.

## Что здесь будет

- команды бота;
- сценарии быстрого бронирования;
- напоминания о бронях;
- запросы к общему API и обработка callback-кнопок.

## Основные папки

- `src/bot` — инициализация бота
- `src/commands` — slash и текстовые команды
- `src/scenes` — пошаговые сценарии
- `src/services` — бизнес-обвязка под bot flow
- `src/lib` — утилиты и хелперы
