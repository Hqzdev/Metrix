# Web App

Это клиентское и серверное приложение на `Next.js`.

## Что здесь будет

- web-интерфейс для сотрудников и администраторов;
- страницы бронирований, ресурсов, аналитики и настроек;
- route handlers для web-слоя, если часть API будет жить в приложении;
- UI-компоненты, hooks и feature-слой.

## Основные папки

- `src/app` — роутинг и страницы
- `src/components` — UI-компоненты
- `src/features` — продуктовые сценарии
- `src/hooks` — кастомные React hooks
- `src/lib` — web-утилиты
- `public` — статические файлы
